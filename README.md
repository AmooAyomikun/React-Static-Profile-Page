#  Static Profile Card
This project showcases a clean frontend architecture using reusable React components, dynamic rendering with props and .map(), responsive layouts, and modern UI styling.

##  The application displays:
1.  Profile banner
2.  Avatar section
3.  Developer bio
4.  Stats section
5.  Skills list
6.  Social media links
7.  Designed to be beginner-friendly while following good React development practices.

##  Preview
https://amooayomikun.github.io/React-Static-Profile-Page/

##  Features
1.  Modern responsive UI
2.  Reusable React components
3.  Dynamic rendering using props
4.  Data-driven rendering using .map()
5.  Responsive Flexbox layouts
6.  Social media cards with Font Awesome icons
7.  Hover animations and smooth UI interactions
8.  Mobile-friendly design
9.  Clean component architecture

##  Built With
* **React** - Component-based architecture
* **JavaScript (ES6+)** - Modern logic and syntax
* **CSS3** - Custom styling and variables
* **Flexbox** - Modern layout system
* **Font Awesome** - Scalable vector icons
* **Vite** - Lightning-fast build tool

## What I Learned
During this project, I strengthened my skills in:
* **React Architecture**: Managing component structure and data flow.
* **Component Communication**: Passing data efficiently via **Props**.
* **Dynamic UI**: Using `.map()` to render interfaces from data arrays.
* **CSS Mastery**: Mastering `absolute` vs `relative` positioning for complex layouts (like the Avatar overlay).
* **Responsive Design**: Ensuring the UI stays safe across different screen sizes.
